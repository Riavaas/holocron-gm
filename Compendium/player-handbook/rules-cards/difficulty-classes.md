# Difficulty Classes

## Quick Rule

Set a DC for static challenges. A roll succeeds when the total meets or exceeds the DC.

## When Used

Use for locks, hazards, research, negotiation pressure, environmental obstacles, and most non-opposed tasks.

## GM Notes

When unsure, use this ruling pattern: easy if trained characters should usually pass, moderate if failure is plausible, hard if success should feel notable.
