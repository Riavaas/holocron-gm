# Tools

## Quick Rule

Tools enable trained tasks and can add proficiency or unlock attempts.

## When Used

Use for crafting, repairs, slicing-like tasks, music, games, and specialist kit work.

## GM Notes

Pair tools with ability checks when the method matters.
