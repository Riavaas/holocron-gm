# Armor Class from Equipment

## Quick Rule

Armor and shields can change Armor Class when worn, held, and used correctly.

## When Used

Use when players equip armor, shields, or defensive accessories.

## GM Notes

Confirm proficiency, armor category, shield state, and any property limits.
