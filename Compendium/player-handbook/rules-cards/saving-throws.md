# Saving Throws

## Quick Rule

Roll d20 + ability modifier + save proficiency if applicable against the effect's DC.

## When Used

Use when a character resists danger, a power, a trap, a condition, or another harmful effect.

## GM Notes

If the character initiates the attempt, consider an ability check. If danger acts on the character, call for a save.

SW5e Player's Handbook, page 214.
