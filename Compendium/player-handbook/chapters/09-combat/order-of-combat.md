# Order of Combat

## Source

* Book: SW5e Player's Handbook
* Pages: 221-222
* Original section: The Order of Combat

## Summary

Combat begins by establishing positions, surprise, and initiative. Rounds cycle through turns until the scene ends.

## Key Rules

* Determine surprise when one side is unaware or unprepared.
* Roll initiative to set turn order.
* A round represents a short shared slice of action.
* On a turn, a creature can move and take one action, with bonus actions and reactions only when allowed.
* Reactions occur in response to triggers and refresh by turn timing.

## GM Use

Start combat cleanly: map positions, decide surprise, roll initiative, then run turns in order.

## Common Rulings

* Surprise is about awareness, not whether players expected a fight.
* Bonus actions require a feature, power, or rule that grants one.
* Reactions need a trigger; no trigger, no reaction.
