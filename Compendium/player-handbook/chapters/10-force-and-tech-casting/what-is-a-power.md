# What Is a Power

## Source

* Book: SW5e Player's Handbook
* Pages: 231-231
* Original section: What Is a Power

## Summary

Powers are structured Force or tech effects with level, timing, range, duration, and targeting rules.

## Key Rules

* Powers use a shared rules framework.
* Power level affects cost and strength.
* Known powers define what a character can cast.
* At-will powers and leveled powers behave differently.

## GM Use

Use this as the entry point whenever a player asks whether a power can solve a scene problem.

## Common Rulings

* If the power description is clear, follow it.
* If unclear, apply casting time, target, range, and duration first.
