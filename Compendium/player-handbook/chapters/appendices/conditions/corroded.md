# Corroded

## Quick Effect

SW5e condition tied to acid/corrosion effects and gear/body degradation.

## Mechanical Effects

* Track the source effect for exact penalty.
* Use it as a flag for ongoing corrosive consequences.
* Apply saving throw or duration from the causing feature.
* Track duration, source, save timing, and removal condition in the combat tracker.

## GM Use

Apply this as a visible status marker. On each affected creature turn, check whether the condition changes movement, action options, attacks, saves, or perception.
