# Dropping to 0 Hit Points and Death

## Source

* Book: SW5e Player's Handbook
* Pages: 228-229
* Original section: Dropping to 0 Hit Points

## Summary

When a creature drops to 0 hit points, the rules determine whether it dies instantly, falls unconscious, makes death saving throws, stabilizes, or is knocked out.

## Key Rules

* Massive damage can cause instant death if the overflow is severe enough.
* A player character at 0 HP often falls unconscious and makes death saving throws.
* Stabilizing stops death save progression without restoring combat readiness.
* Monsters often die at 0 HP unless the GM chooses otherwise.
* A melee attacker can choose to knock a creature out when appropriate.

## GM Use

Pause at 0 HP. Ask whether the damage instantly kills, whether the attacker knocks out, and whether death saves or monster death rules apply.

## Common Rulings

* Named NPCs can use PC-style 0 HP rules when dramatic.
* Healing at 0 HP should be resolved before the creature's next death save.
* Knockouts should require the attacker to be in a position to control the finishing blow.
