# Force and Tech Points

## Source

* Book: SW5e Player's Handbook
* Pages: 231-231
* Original section: Force and Tech Points

## Summary

Force and tech points are casting resources spent to use powers and scale effects.

## Key Rules

* Track the correct point pool.
* Higher-level casting can cost more resources.
* At-will powers do not use the same spend pattern.

## GM Use

Ask players to announce the power level and resource spend before resolving effects.

## Common Rulings

* If the player lacks points, the power cannot be cast at that level.
* If a feature modifies cost, note it before the roll.
