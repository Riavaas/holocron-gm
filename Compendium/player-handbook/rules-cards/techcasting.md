# Techcasting

## Quick Rule

Techcasting uses tech powers through a character's techcasting rules and ability.

## When Used

Use when a character casts a tech power or a feature references tech powers.

## GM Notes

Check known powers, tech points, casting ability, save DC, attack modifier, and devices or focus requirements.
