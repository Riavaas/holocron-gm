# Tools

## Source

* Book: SW5e Player's Handbook
* Pages: 182-182
* Original section: Tools

## Summary

Tools represent trained equipment sets for crafting, repair, art, gaming, music, and specialist tasks.

## Key Rules

* Tool proficiency can apply to relevant checks.
* Specialist kits enable tasks that bare hands cannot.
* Tools can overlap with skills when both fit the method.

## GM Use

Ask what tool is used and what the character is trying to accomplish.

## Common Rulings

* Let tool proficiency lower DC, add proficiency, or unlock the attempt depending on the task.
