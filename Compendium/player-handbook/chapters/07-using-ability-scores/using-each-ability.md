# Using Each Ability

## Source

* Book: SW5e Player's Handbook
* Pages: 210-214
* Original section: Using Each Ability

## Summary

Each ability covers a different mode of action: force, agility, endurance, analysis, awareness, and influence. The chapter gives common check types and derived uses for combat, casting, initiative, hit points, and defenses.

## Key Rules

* Strength covers athletics, lifting, carrying, forced movement, and raw physical effort.
* Dexterity covers stealth, acrobatics, initiative, many attacks, and Armor Class.
* Constitution covers endurance, concentration-like stamina calls, and hit point durability.
* Intelligence covers lore, investigation, technical reasoning, and techcasting ability.
* Wisdom covers perception, insight, survival, and forcecasting ability for some characters.
* Charisma covers deception, persuasion, intimidation, performance, and forcecasting ability for some characters.

## GM Use

Pick the ability from the character's approach, not the outcome they want. A locked door can be Strength to force, Dexterity to bypass, Intelligence to analyze, or Charisma to convince someone to open it.

## Common Rulings

* Let players pitch alternate abilities, then require a matching method.
* For physical hazards, decide whether the issue is power, reflex, or endurance.
* For social scenes, decide whether the character lies, bargains, threatens, or performs.
