# Short Rest

## Quick Rule

A short rest is a brief downtime break that can restore limited resources and allow hit dice recovery if conditions permit.

## When Used

Use between encounters when the party can pause but not fully reset.

## GM Notes

Ask where the rest happens, who keeps watch, and whether enemies have time to react.
