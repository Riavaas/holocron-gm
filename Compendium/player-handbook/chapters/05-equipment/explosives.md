# Explosives

## Source

* Book: SW5e Player's Handbook
* Pages: 177-177
* Original section: Explosives

## Summary

Explosives are high-impact consumables such as grenades, mines, and detonators.

## Key Rules

* Confirm type, placement or throw, trigger, range, area, save or attack, and damage.
* Mines and remote detonation depend on setup.
* Area effects can hit allies and objects.

## GM Use

Use explosives as tactical scene changers. Make blast zones visible before resolution when characters can judge them.

## Common Rulings

* If placement is rushed, call for a check.
* If timing matters, use initiative, Ready, or object interaction logic.
