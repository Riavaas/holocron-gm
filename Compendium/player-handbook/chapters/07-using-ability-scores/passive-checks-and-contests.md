# Passive Checks and Contests

## Source

* Book: SW5e Player's Handbook
* Pages: 208-210
* Original section: Contests, Passive Checks

## Summary

Passive checks represent average effort without rolling. Contests resolve direct opposition by comparing checks from two sides.

## Key Rules

* Passive checks use a fixed check total, commonly 10 plus modifiers.
* Passive checks are useful for hidden information and repeated tasks.
* A contest compares checks from opposing creatures.
* The higher contest result wins; ties keep the situation unchanged unless the GM rules otherwise.

## GM Use

Use passive checks to avoid alerting players that hidden information exists. Use contests when two creatures are directly pushing against each other.

## Common Rulings

* Passive Perception is the default for noticing hidden threats.
* Contests work well for grapples, shoves, lies versus insight, and races.
* If a tie is awkward, preserve the current state.
